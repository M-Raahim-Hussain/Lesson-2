HTML5

HyperText Markup Language is the standard markup language for creating web pages. It describes the structure of a Web page semantically and originally included cues for the appearance of the document.

Combined with CSS and JavaScript, it forms the "holy trinity" of front-end development, allowing for rich, interactive experiences.

CSS3

Cascading Style Sheets is the design language of the web. It allows you to transform plain HTML into beautiful, responsive interfaces using layouts, colors, and animations.

GitHub

GitHub is the world's leading AI-powered developer platform. It hosts millions of projects, allowing teams to collaborate on code, track versions, and automate workflows.
